CV Maker

Secret side project to help my latest venture. I built this application for the MVP stage and present it to the company's board of directors. Currently, the project has been validated and has entered the beta testing phase by a small group of users Production launch planned for the first quarter of 2020

What's the problem ?

Let me add some context. This firm is a construction company whith public authorty as biggest client. French legal framework is really strict, companies needs to allocate perfect qualified human ressources on public projects. So, to prove it, public work firms needs to send dedicated human ressouces resumes withing the project tender. Resume need to focus on experiences with comparable projects. It's not a resume for job seeking. Actually, every tender maker build it on his laptop each time he needs it. No data sharing, no consistency.

What's the solution ?

This app generate perfectly fresh resume for selected team in less thas 4 steps. Users can add fresh experiences to a worker's profile is 2 click. Sometimes some information is missing, or partially missing, and there is a system of data confidence (red, orange, green flag) to prevent the recording of bad data. App is backed by database.
